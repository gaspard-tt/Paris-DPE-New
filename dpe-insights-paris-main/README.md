# Mon DPE — Simulateur de Performance Énergétique

Estimez la classe DPE de votre logement, identifiez ses points faibles et recevez des recommandations de rénovation personnalisées.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- React Router
- React Query

## Getting started

```sh
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server on port 8080 |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
| `npm run test` | Run tests |
